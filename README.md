# Cryptocurrency Tracker 💰📈

A web application to track live cryptocurrency data with auto-update features and interactive user experience.

## Tech Stack
- HTML5 🌐
- CSS3 🎨
- JavaScript ⚙️
- CoinCap API 🔗
- Tawk.to (Live Chat) 💬

## Features
- Live Cryptocurrency Prices 📊
- Auto Data Refresh Every 30 Seconds 🔄
- 50+ Cryptocurrency Support 🔥
- Mobile & Desktop Responsive 📱💻
- Live Chat Support 💬
- Real-time Market Trends


## 🚀 Installation Steps
1. Clone the Repository:
git clone https://github.com/abhishek.kr23/CryptoTracker.git
cd CryptoTracker

2. Open index.html in your browser
Open with Live Server (Optional)
